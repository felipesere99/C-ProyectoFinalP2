Detailed Description

La clase TableroPrinter es bastante autoexplicativa, es una clase que utlizando como base la interfaz IPrinter, se encarga de implimir lo creado en la clase tablero. La clase implementa la interfaz IPrinter y proporciona implementaciones específicas para los métodos ImprimirTableroConBarcos e ImprimirTableroConDisparos. Esto sigue el principio de ISP al asegurarse de que las clases no estén obligadas a implementar métodos que no utilizan. La clase tambien tiene la responsabilidad de imprimir tableros, pero no de gestionar la lógica del juego ni de almacenar la información del tablero, esto sigue el SRP.
